# movie_review

## サイト概要
### サイトテーマ
他人の書いた映画のレビューを見たり、自分で投稿したりできるコミュニティサイト

### テーマを選んだ理由
映画を見る際に、よくレビューサイトを活用するのですが、その際にこんな機能があったら便利なのにと思うことがあったため

### ターゲットユーザ
映画を見るのが好きな人

### 主な利用シーン
「この映画を見ようかな？」と悩んでいるときに他人の投稿を見て判断できる
